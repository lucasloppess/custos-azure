# custos-azure
Otimizando Custos no Azure
TCO - Custo total de Propriedade

https://azure.microsoft.com/pt-br/pricing/tco/calculator/ - Estime a economia de custos gerada pela migração de cargas de trabalho para o Azure
Calculadora possui alguns modelos de cargas de trabalho que serão enviadas para nuvem.

Após preencher tudo, ele te da um relatório em relação ao ambiente Onpremisses.

Fazendo uma simulação na TCO é o melhor cenário para apresentação ao cliente.

Calculadora de preço - Calcule os custos estimados por hora ou mensais para usar o Azure. Cenário diferente da TCO, utilizado mais por empresas.

https://azure.microsoft.com/pt-br/pricing/calculator/

Cost management - Você consegue verificar quanto está gastando no Azure.
